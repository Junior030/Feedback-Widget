#### Ambiente linux
<br>

```bash
  
  git clone https://github.com/Junior030/Feedback-Widget.git

  cd Feedback-Widget
  
  npm run start

``` 
#### Outros ambientes 
<br>

* Back-end
```bash
  git clone https://github.com/Junior030/Feedback-Widget.git

  cd Feedback-Widget/server

  npm run dev

```
<br>

  * Front-end
```bash
  git clone https://github.com/Junior030/Feedback-Widget.git

  cd Feedback-Widget/web

  npm run dev

```
